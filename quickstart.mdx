## Get started in three steps

### Step 1: Install and run Beekeepr

1. Clone the Beekeepr repository.  
2. Install dependencies with `npm install`.  
3. Start the server with `npm run dev`.  

### Step 2: Configure

1. Create a `.env` file in the root directory.  
2. Add your database connection and required keys.  
3. Save and restart the server.  

### Step 3: Use Beekeepr

1. Open `http://localhost:8080` in your browser.  
2. Create your first user and log in.  
3. Start managing your beehives and metrics.
